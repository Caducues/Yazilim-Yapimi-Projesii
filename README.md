Yazilim-Yapimi-Projesi
Bu program, kullanıcıların İngilizce ve Türkçe kelimeleri resimlerle birlikte girmelerine olanak tanır. Kelimeler, kullanıcı tarafından altı kez doğru yanıtlandığında,
belirli zaman aralıklarında (1 gün, 1 hafta, 1 ay, 6 ay, 1 yıl) tekrar sorularak öğrenilmesi sağlanır. Her seferinde doğru cevaplanan kelimeler öğrenilmiş kabul edilir.

Projeyi Yükleme:

-Projeyi yüklemeden önce -sql jdbc 12.6.1.0 eklentisini indirmeniz gerekmektedir.(Java ile sql serverı birbirine bağlamak için gerekli) 
-Projede Java JDK 22.0.1 kullanılmıştır.
-Project structure ayarlarından SDK Yolunu değiştirmeniz gerekiyor. -Projede MSSQL 2022 kullanılmıştır Database klasöründeki backup dosyası yardımıyla MSSQLe gerekli databasei kurabilirsiniz.
